# entity-cleanup

A small script to cleanup nearby entities

## Dependencies

- The server must be running `onesync`

## Features

- Simple command `/cleanup`
- Supports
	- [x] networked entities
	- [ ] Unetworked entities
- Deletes
	- [x] Vehicles
	- [ ] Peds
	- [ ] Objects

## Usage

In `chat` or in the `F8` console use `/cleanup [vehicle | ped | object]`
